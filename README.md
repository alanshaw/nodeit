# nodeit [![experimental](http://hughsk.github.io/stability-badges/dist/experimental.svg)](http://github.com/hughsk/stability-badges)

> NodeJS text editor

## Getting started

Install browserify:

```sh
npm install -g browserify
```

Build:

```sh
browserify js/main.js -o js/bundle.js
```
